## 1. 这本书在讲什么

这本书的主线不是“再列几个有效因子”，而是系统回答：

- 各类资产的长期预期收益从哪里来；
- 风险补偿、行为偏差、制度摩擦和估值回归分别扮演什么角色；
- 历史平均收益为什么不能直接当成未来收益；
- 如何把估值、利差、宏观、调查、趋势、波动率和流动性等变量统一进一个前瞻收益框架；
- 在知道“预期收益可能在哪”之后，怎样通过控制风险、期限、技能和成本，把纸面收益尽可能保留下来。

它比一般股票因子书更广：

- 不只讲股票；
- 也系统覆盖债券、信用、外汇、商品、波动率、尾部风险、通胀与流动性；
- 既讲长期均值，也讲条件化收益；
- 既讲收益来源，也讲实施约束。

如果说 `quant_factor.md` 更像“量化因子与组合实现规范”，这本书更像“**跨资产 expected return engine 的研究母本**”。

## 3. 全书最重要的总主线

### 3.1 历史均值不是未来收益

作者显然反复强调：

- 历史均值是一个起点；
- 但未来收益会受当前估值、风险厌恶、流动性状态、通胀、增长和制度环境影响；
- 所以预期收益必须是条件化的，而不是静态常数。

统一抽象：

```python
expected_return_t = long_run_mean + state_adjustment_t
```

其中 `state_adjustment_t` 可以进一步拆为：

```python
state_adjustment_t = (
    valuation_component_t
    + macro_component_t
    + spread_component_t
    + liquidity_component_t
    + sentiment_component_t
)
```

### 3.2 收益来源不只一种

全书结构本身就在表达：

1. 有些收益是风险补偿；
2. 有些收益更像行为偏差；
3. 有些收益来自制度摩擦或资本约束；
4. 有些收益带有明显的灾难保险卖方特征；
5. 有些收益高度依赖初始估值和市场环境。

所以做多策略组合时，不应只问“收益高不高”，还应问：

- 它赚的是哪一类钱；
- 它在危机时怎么亏；
- 它和其他策略是否共享同一风险源。

### 3.3 跨资产比较是稳健性检验

这本书很重要的一点是：

- 价值不只看股票；
- carry 不只看外汇；
- 趋势不只看商品；
- 波动率风险溢价、流动性溢价、尾部风险也都在跨资产语境下讨论。

这使它特别适合训练一种思维：

- 真正稳健的收益源，往往能在多个市场以不同形式复现；
- 只在单一市场单一时期成立的结论，要格外警惕过拟合和制度特例。

#### 章节主题

- 12.1 简介

章节首页和相关页能确认：

- 讨论价值策略的长期证据；
- 也讨论价值策略与“廉价但差公司”之间的区分问题；
- 讨论价值与成长之间并不是简单对立。

#### 核心思想

本章不是只说“低估值有效”，而是系统处理四个问题：

1. 价值到底用什么量化；
2. 价值溢价的长期证据有多稳；
3. 价值为什么有效；
4. 价值是不是只是在买入烂公司、卖出好公司。

从 OCR 恢复内容看，作者显然引用了 DFA、Ken French 数据和长期样本，说明其论证方式偏向：

- 长期横截面证据；
- 因子组合表现；
- 不同估值口径之间的对照；
- 价值与成长、质量之间的关系。

#### 可提炼信号

1. **BM / Book-to-Market**
   ```python
   bm_t = book_equity_t / market_cap_t
   ```
2. **EP / Earnings-to-Price**
   ```python
   ep_t = earnings_ttm_t / market_cap_t
   ```
3. **CFP / Cashflow-to-Price**
   ```python
   cfp_t = cashflow_ttm_t / market_cap_t
   ```
4. **Dividend Yield**
   ```python
   dp_t = dividend_ttm_t / market_cap_t
   ```
5. **EV 口径价值**
   ```python
   cfo2ev_t = cfo_ttm_t / enterprise_value_t
   ```

#### 实现提示

- 价值信号适合中长期 horizon，不适合高频更新；
- 必须处理财报滞后和 point-in-time；
- 价值应与质量结合，否则容易混入财务困境暴露；
- 价值与成长不是非此即彼，更合理的是把“便宜程度”和“基本面质量/成长持续性”拆开建模。

### 4.2 第 13 章 货币利差策略

#### 章节主题

目录可确认本章至少包括：

- 13.1 简介
- 13.2 历史平均收益
- 13.3 基本本利差策略的改进和检测
- 13.4 为什么利差策略有效
- 13.5 分歧的相关交易

OCR 可读页明确出现：

- G10；
- 历史收益图；
- Bloomberg、Citigroup 等数据源；
- 改进版本与相关交易。

#### 核心思想

本章讨论的不是“外汇预测汇率”，而是更经典的：

- 利率高的货币与利率低的货币之间，存在长期 carry 收益；
- 但这种收益常伴随危机时的集中回撤；
- 基础 carry 可以进一步加入估值、波动率、风险厌恶或趋势过滤来改进。

#### 可提炼信号

1. **基础 carry**
   ```python
   fx_carry_t = short_rate_high_ccy_t - short_rate_low_ccy_t
   ```
2. **远期贴水/升水代理**
   ```python
   forward_discount_t = forward_t / spot_t - 1
   ```
3. **风险调整 carry**
   ```python
   carry_score_t = carry_t / vol_t
   ```
4. **carry + valuation**
   - 在 PPP 明显偏离时，修正 carry 的仓位权重；
5. **carry + trend**
   - 当趋势方向与 carry 一致时提高权重。

#### 实现提示

- 统一货币对方向，否则符号会错；
- 收益需拆成利差收益、汇率变动收益和滚动收益；
- 危机时 carry 往往与风险资产同步受损，不能把它当无风险套利。

### 4.3 第 14 章 大宗商品动量和趋势跟踪策略

1. **时间序列动量**

2. **横截面商品动量**

### 4.4 第 15 章 出售（股指）波动性策略

#### 核心思想

这章的核心不是“卖波动率收益高”，而是：

- 卖波动率赚的是保险费；
- 保险费长期偏贵，导致平均收益为正；
- 但这种收益具有明显负偏度；
- delta 对冲并不能消除所有风险，尤其在跳跃和波动率爆发时。

#### 可提炼信号


   ```python
   variance_risk_premium_t = implied_var_t - realized_var_t
   ```
3. **卖保护性期权组合的 carry**
4. **delta-hedged option PnL**
   ```python
   option_carry_pnl_t = option_theta_t + vol_mispricing_t - hedge_cost_t
   ```

#### 实现提示

- 不能只看均值和夏普，要看偏度、峰度和灾难期损失；
- 保证金占用、流动性和跳空风险是核心；
- 这类策略在组合里更像“持续收租 + 偶尔遭遇大额赔付”。

### 4.7 第 18 章 流动性因素和低流动性溢价

#### 核心思想

这里讨论的不是“流动性差所以收益高”这么简单，而是：

- 流动性自身是一个时变状态变量；
- 低流动性资产长期可能提供补偿；
- 但在流动性收缩时，这种补偿往往伴随更大的回撤；
- 因而“流动性溢价”与“流动性崩塌风险”是同一枚硬币的两面。

#### 可提炼信号

1. **bid-ask spread**
2. **ADV / volume participation**
3. **Amihud illiquidity**
4. **Pastor-Stambaugh 流动性暴露**
5. **funding liquidity proxy**

#### 实现提示

- 要区分个券流动性和市场整体流动性；
- 流动性差异既能解释收益，也会限制可实现容量；
- 这章的结论非常适合转化为容量约束和成本模型，而不仅是 alpha。

#### 核心思想

## 7. 这本书对股票量化最有价值的启发

### 7.1 价值必须条件化理解

这本书强化了一个观点：

- 价值不是孤立横截面排序；
- 它是“价格相对长期锚的偏离”；
- 偏离越大，长期预期收益越高，但实现时间和路径高度不稳定。

### 7.2 风险卖方收益要看尾部

无论是信用、波动率、低流动性，还是某些 carry 策略，本质上都可能在卖尾部保险。

所以：

- 高平均收益不必然代表高质量 alpha；
- 必须看偏度、回撤、危机期相关性和再融资压力。

### 7.3 经济周期与风格收益联立建模

从第 26 章可见，风格收益会随宏观状态变化。

这直接启发：

- 因子配置不应只看静态历史均值；
- 可以把 value / momentum / size / quality 放进 regime-switching 框架。

### 7.4 长期资金的优势是一种 alpha 放大器

第 28～29 章很强调：

- 投资期限；
- 再平衡纪律；
- 低成本；
- 不被短期排名驱动；

本身就能提高可实现收益。

也就是说，投资流程本身也是 alpha 的一部分。
