# 参考书目录说明

本目录统一存放量化研究相关的原始参考书 PDF，以及对应的阅读总结 / OCR 整理 / 方法论提炼文档。

## 使用约定

- 新增参考书时，原始 PDF 与对应总结文档优先一起放在本目录。
- 如果一本书存在多语言版本，可以保留多份总结，但请在本表中明确区分。
- 如果一本原始资料对应多份专题笔记，也请在本表补充映射关系，避免总结文档再次散落到仓库根目录。
