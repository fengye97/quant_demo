## 第一层：因子投资理论基石（理解因子从哪来）

### 《主动投资组合管理》— Richard C. Grinold & Ronald N. Kahn

- **地位**：量化组合投资领域「圣经」
- **中文版**：机械工业出版社，2014 年初版 / 2025 年典藏版，李腾 / 杨柯敏 / 刘震译
- **为什么是基石**：提出信息比率 **IR = IC × √BR** 基本法则，定义了信息系数(IC)、广度(BR)、转移系数(TC)三维评估模型。每挖掘一个候选因子，都需要这套框架来判断它是否值得进入模型。
- **核心概念**：信息比率(IR)、信息系数(IC)、策略广度(BR)、转移系数(TC)、主动管理基本法则、BARRA 风险模型

### 《预期收益：投资者获利指南》— Antti Ilmanen（AQR 合伙人）

- **地位**：因子收益来源「百科全书」
- **中文版**：格致出版社 / 上海人民出版社，2018 年，钱磊译
- **2024 续作**：《预期收益：在不确定市场创造非凡回报》（Investing Amid Low Expected Returns），朱俊磊译，格致出版社
- **为什么是百科全书**：一站式覆盖股票、债券、另类资产的价值因子、动量因子、利差因子、波动率因子、流动性因子、尾部风险因子，以及背后的宏观因子（增长、通胀、流动性）。直接给出一份完整的因子分类地图。
- **覆盖因子类别**：价值(Value)、动量(Momentum)、利差(Carry)、低波动(Low Volatility)、流动性(Liquidity)、尾部风险(Tail Risk)、宏观因子(增长/通胀/流动性)

### 《量化股票组合管理》— Ludwig B. Chincarini & Daehwan Kim

- **地位**：量化组合落地「工程手册」
- **中文版**：机械工业出版社，2018 年初版 / 2025 年典藏版，陈志岗 / 李腾译
- **豆瓣评分**：8.8 分
- **为什么是工程手册**：因子筛选 → 基本面因子模型 → 经济因子模型 → 股票排序 → 组合权重 → 再平衡 → 交易成本 → 税务管理 → 市场中性 → 贝叶斯 α → 业绩归因 → 回测流程。完整的工程落地链路。

## 第二层：A 股因子实证（验证因子在 A 股是否有效）

### 《因子投资：方法与实践》— 石川 / 刘洋溢 / 连祥斌 ← A 股因子研究首选

- **为什么最重要**：目前市面上唯一一本**系统性地在 A 股上做独立因子实证**的书。涵盖主流因子解读（动量、波动率、价值、质量等）、多因子模型构建、异象研究，以及行为金融学、机器学习与因子投资的结合。国际因子书告诉你「什么因子在美国有效」，这本书告诉你「什么因子在中国有效」。

### 《基本面量化投资策略：实证检验 A 股常用 53 个单因子和 31 个多因子策略》— 董鹏飞

- **作者背景**：央企资管副总裁，前易方达/中信证券
- **回测数据**：1998 年底至 2020 年底共 22 年 A 股数据
- **为什么实战性最强**：53 个单因子逐一回测——市值、市盈率、市净率、企业价值倍数、市销率、市现率、股息率、成长性、ROIC/ROE、利润率、动量/反转/波动率等。31 个多因子组合策略。危险信号筛选体系。神奇公式的中国实证。
- **相当于**：一份经过 22 年数据验证的 A 股因子清单。

### 《基本面量化投资 18 种经典策略》— 张然（青年长江学者）/ 汪荣飞

- **为什么学术+实践结合最好**：每章分「文献导读」+「中国 A 股实践」两部分。18 个主题对应 18 类选股因子：
  - 公司价值（价值因子）
  - 盈利能力 / 经营效率（杜邦分析）
  - 盈余质量 / 投资与融资
  - 无形资产 / 研发创新
  - 财务安全 / 证券分析师
  - 机构投资者 / 内部人交易
  - 卖空策略
  - **价格动量** / **市场情绪**
  - **文本分析 / NLP**

### 《股票量化投资策略：基于中国 A 股市场的新发现》— 朱一峰（2025.02 新书）

- **作者背景**：中央财经大学，Emory 大学经济学博士
- **为什么前沿**：涵盖较新的 A 股因子研究方向——
  - 在险价值(VaR) / 预期亏损(ES)尾部风险因子
  - 趋势因子 / 凸显理论
  - 分位数策略
  - 融资融券制度对波动率策略的影响
- **获世界计量经济学会院士查涛教授高度评价**

### 《因子模型、机构投资者与资本市场定价效率》— 李志勇（2025.01 新书）

- **核心主题**：
  - 「因子动物园」(Factor Zoo)
  - 因子择时理论与实证
  - 中国壳价值(Shell Value)与因子模型修正
  - 公募基金收益率可预测性
  - 可转债市场因子模型
  - **机器学习在资产定价中的应用**

## 第三层：技术体系量化（把图表变成因子）

### 缠论系列 — 最丰富的 A 股本土技术因子源

缠论由已故传奇博主"缠中说禅"创立，以「走势必完美」为哲学基石，构建了包含分型、笔、线段、中枢、背驰、级别等概念的完整技术分析体系。

**为什么特别适合量化因子挖掘**：相比道氏/波浪等理论，缠论的概念天然可量化——

| 概念 | 量化方式 |
|------|---------|
| 分型 | 局部极值点识别（顶分型/底分型），可直接写成代码 |
| 笔 | 经过包含处理后的相邻分型连线，可程序化的趋势段 |
| 线段 | 至少三笔构成，定义明确可编程 |
| 中枢 | 至少三段次级别走势重叠，本质是「价格密集区」 |
| 背驰 | 趋势力度衰竭信号，可量化为 MACD 面积/力度对比 |
| 区间套 | 多级别联立的精确定位方法 |
| 三类买卖点 | 可量化的入场/离场规则 |
