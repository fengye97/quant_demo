## 5. 第三方库安装

### 5.1 akshare

- **版本**: 1.18.60 (最新)
- **安装方式**: `python3 -m pip install akshare --no-deps` + 手动安装缺失依赖
- **已安装依赖**: requests, lxml, beautifulsoup4, xlrd, openpyxl, pandas, tqdm, jsonpath, curl_cffi, decorator, html5lib, mini-racer, tabulate
- **验证**: 成功通过 `akshare.stock_zh_a_hist(symbol='000001', period='daily', start_date='20260501', end_date='20260515')` 获取平安银行8个交易日数据（OHLCV + 振幅 + 涨跌幅 + 换手率）
- **已知限制**: Python 3.9 (Homebrew) 有pip弃用警告；`aiohttp>=3.11.13` 与现有包冲突，使用 `--no-deps` 绕过。核心数据获取功能完全正常。
- **建议**: 如需完整功能（websocket实时数据等），在干净conda环境中重装

### 5.2 tushare

- **版本**: 1.4.29

- **适用场景**: 基本面数据、财务数据、指数成分等（补充当前Sina API覆盖不到的维度）
